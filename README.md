dotfiles
========

## はじめにすること

1. oh-my-zshのインストール
2. neobundleのインストール
3. 以下のコマンドを実行

```
ln -s ~/dotfiles/.zshenv ~/.zshenv
ln -s ~/dotfiles/.zshrc ~/.zshrc
ln -s ~/dotfiles/.vimrc ~/.vimrc
ln -s ~/dotfiles/.tmux.conf ~/.tmux.conf
ln -s ~/dotfiles/muse-with-hostname.zsh-theme ~/.oh-my-zsh/custom/muse-with-hostname.zsh-theme
```
